# Таблица сотрудников: данные, сортировка, поиск

Интерактивная таблица с данными о сотрудниках компании. Проект постепенно развивается.

## Описание

Веб‑приложение отображает таблицу с информацией о сотрудниках:
* ФИО сотрудника;
* должность;
* размер заработной платы.

Реализованы базовые функции для удобной работы с данными:
* **поиск** по ФИО и должности;
* **сортировка** по столбцам (ФИО, должность, зарплата);
* **фильтрация** данных (например, по диапазону зарплат или по отделам).

Проект создан для демонстрации работы с табличными данными в веб‑интерфейсе и может быть использован как основа для HR‑систем или внутренних корпоративных порталов.

## Функциональность

На текущем этапе реализованы:

* отображение данных в виде таблицы с разбивкой по столбцам;
* сортировка по клику на заголовок столбца (по возрастанию/убыванию);
* строка поиска с мгновенной фильтрацией результатов по мере ввода;
* фильтр по диапазону заработной платы (минимум–максимум);
* адаптивный дизайн — корректное отображение на мобильных устройствах и десктопах.
